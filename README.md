# iSujou

Aplicativo web e mobile com intuito de realizar a ligação entre empregados domésticos e proprietários de imóveis, facilitando a comunicação e fechamento de contratos para limpeza.

## Tecnologias
- .NET Core 3.1
- React
- React Native
- SQL Server
